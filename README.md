# HT Task Manager - Frontend

This repository contains the **frontend** for the HM Task Manager (HMTM) application.  
Follow the steps below to set up and run the frontend in your development environment.

---

## Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/) (optional, can use npm)

---

## Setup Instructions

### 1. Install Dependencies

Install all required packages using either Yarn or npm:

```bash
# Using Yarn
yarn install

# OR using npm
npm install
```

### 2. Start Development Server

Run the frontend in development mode:

```bash
# Using Yarn
yarn dev

# OR using npm
npm run dev
```

The HM Task Manager Frontend should now be running at http://localhost:5173
